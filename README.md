# ttrpg.co
A WIP Table-Top Role-Playing-Game (TTRPG) Service

### Installation

To run the server, use the following:

`cd server`
`npm install`
`npm run`

Join on localhost:8080

### Development

To run the server with nodemon (for dev purposes), use the following:

`cd server`
`npm install --dev`
`npm run dev`

Join on localhost:8080
